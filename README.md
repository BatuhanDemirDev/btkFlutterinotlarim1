# btkakademi

Bu proje btkakademi kısmındaki çalışma notlarım ve ilerleyişim için oluşturulmuştur.

- Açıklama: btkakademi kısmındaki çalışma notlarım ve ilerleyişim
- Başlangıç dosyası: lib/main.dart (örnek uygulama içerir)

Nasıl çalıştırılır:

```bash
cd btkakademi
flutter pub get
flutter run
```
